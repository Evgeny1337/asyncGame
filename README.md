
Асинхронная анимация космического пространства с управляемой ракетой и мерцающими звёздами, написанная на Python с использованием библиотеки `curses`.

##  Особенности

-   **Управляемая ракета** с анимацией двигателей (2 кадра анимации)
    
-   **Мерцающие звёзды** со случайной анимацией (тусклые/яркие фазы)
    
-   **Асинхронное выполнение** корутин для одновременной анимации объектов
    
-   **Обработка клавиатуры** для управления ракетой
    
-   **Границы экрана** с проверкой столкновений
    

## Управление

-   **Стрелки (↑ ↓ ← →)** - движение ракеты
    
-   **Пробел** - выстрел (в разработке)
    

## Архитектура

### Основные компоненты

1.  **`animate_spaceship()`** - корутина для анимации и управления ракетой
        
2.  **`blink()`** - корутина для анимации звёзд
        
3.  **`fire()`** - корутина для анимации выстрелов (в разработке)
    
4.  **`draw_frame()`** - утилита для отрисовки многострочных фреймов
        
5.  **`read_controls()`** - обработка ввода с клавиатуры
    



## Технические детали

### Временные интервалы

-   **Тик анимации**: 0.1 секунды
    
-   **Смена кадра ракеты**: каждые 2 тика (0.2 секунды)
    
-   **Случайная длительность фаз звёзд**: 1-20 тиков
    

### Ограничения экрана

-   Автоматический расчет границ с учетом размера ракеты
    
-   Ракета не может выйти за пределы экрана
    
-   Учет рамки окна (`border()`)
    

### Асинхронная модель

-   Каждая анимация - независимая корутина
    
-   Центральный event loop управляет всеми корутинами
    
-   Кооперативная многозадачность через `await asyncio.sleep(0)`
    

## Структура файлов

```text

step_2.txt           # Второй кадр ракеты
```
## Запуск

```bash

```
