# Асинхронная аркадная игра про космос с управляемой ракетой и мусором на орбите, написанная на Python с использованием встроенной библиотеки `curses`.

## Особенности

- **Управляемая ракета** с анимацией двигателей (2 кадра) и инерционной физикой
- **Падающий космический мусор** с разными формами из текстовых файлов
- **Мерцающие звёзды** со случайными циклами яркости
- **Информационное окно** с текущим годом и историческими событиями космонавтики
- **Динамическая сложность**: ускорение игры с течением игровых лет
- **Стрельба очередями** по космическому мусору
- **Game Over** при столкновении с мусором

## Управление

- **Стрелки (↑ ↓ ← →)** - плавное движение ракеты
- **Пробел** - стрельба очередями (3 выстрела)

## Архитектура

### Основные корутины

1. **`animate_spaceship()`** - управление ракетой с физикой ускорения
2. **`run_spaceship()`** - контроль стрельбы и проверка столкновений
3. **`fill_orbit_with_garbage()`** - генератор мусора с задержкой по годам
4. **`fly_garbage()`** - анимация падения одного объекта мусора
5. **`blink()`** - анимация мерцания звёзд
6. **`fire()`** - анимация выстрела с проверкой попаданий
7. **`year_counter()`** - игровой таймер, увеличивающий год
8. **`draw_subwindow()`** - информационное окно в правом нижнем углу
9. **`show_gameover()`** - финальная заставка

## Игровая механика

### Прогрессия по годам
- Каждые 40 кадров (4 секунды) год увеличивается на 1
- Частота появления мусора растёт согласно историческим датам
- Скорость игры увеличивается с годами (множитель 1.0 → 2.0)

### Информационное окно
- Отображает текущий год
- Показывает исторические события из космонавтики
- Индикатор текущего множителя скорости

### Система стрельбы
- Очереди по 3 выстрела с задержкой между ними
- Выстрелы уничтожают мусор с анимацией взрыва
- Подсветка уничтоженных препятствий

## Технические детали

### Временные интервалы
- **Тик анимации**: 0.1 секунды (10 FPS)
- **Смена года**: 40 тиков (4 секунды)
- **Задержка мусора**: 20→2 тика (уменьшается с годами)
- **Фазы звёзд**: 1-20 тиков (случайно)

### Ограничения
- Максимум 5 объектов мусора одновременно
- Ракета ограничена рамкой окна с учётом border
- Звёзды не генерируются в области информационного окна

### Асинхронная модель
- Каждый объект - независимая корутина
- Кооперативная многозадачность через `await sleep(1)`
- Центральный event loop управляет всеми корутинами
